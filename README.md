# Программа рисования фракталов

Программа на Python, которая рисует фракталы с использованием заданных пользователем данных.

## Использование

1. Введите данные фрактала:
    - `length` - длина начального отрезка
    - `angle` - угол поворота (в градусах)
    - `depth` - глубина рекурсии
    
2. Запустите программу.

3. На экране появится рисунок фрактала согласно указанным параметрам.

# Зависимости
  * matplotlib - библиотека для визуализации данных.
    
# Заключение
Эта программа позволяет создавать уникальные фракталы с помощью настраиваемых параметров. Попробуйте изменять значения и наслаждайтесь красивыми и сложными фигурами, которые генерируются фракталами.
